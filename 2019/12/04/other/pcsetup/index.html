PC最速セットアップ手順

更新日時:2019-12-04 閲覧時間: 1 min 閲覧回数: 前へ

最悪、PCが不具合あるので仕方ないOS再インストールになりました。

この手順なら半日で開発環境の構築も含めて終わらせることができそう。

バックアップ

  • ISOイメージ作成、USBに焼く
  • フォルダバックアップ（フォルダ、Tools、IEブックマーク）

インストール

USBからWindows10をクリーンインストール

ドメイン未参加の状態で実施する作業

  • PC名変更（機器管理番号）
  • MS Store設定 自動更新Off,タイル製品更新Off
  • MS StoreからLenovoアプリインストール、更新
  • Windows Update
  • Windows機能インストール
    • .NET Framework 3.5(ドメイン内+Proxyは弾かれる、突破できるけど面倒くさい)
    • Hyper-V
    • WSL
    • SandBox

会社の環境で実施する作業

  • ドメイン参加＋自分を管理者権限に設定
  • プロキシ設定
  • アンチウィルスインストール
  • プリンター設定
  • 無線LAN証明書を入れる
  • 暗号化（社内LANに接続すれば、勝手に暗号化してくれる）

インストーラからインストール

  • Office 2019
  • Visual Studio 2019
    • 拡張機能
      • Web Essentials 2019（Web開発なら必須）
      • GitHub Extension for Visual Studio
      • .NET Portability Analyzer（.net updateの修正対象を分析）
      • Productivity Power Tools 2017/2019(生産性向上)
      • StyleCop（コードのスタイルと一貫性をチェック）
      • Analysis Services
      • Reporting Services
      • Integration Services
  • Teams
  • SQLServer Management Studio
  • Claunch（ランチャー）
  • Docker for Windows
  • Kindle
  • Google日本語入力（変換はCtrl + Spaceを割り当てる）
  • Myrica(プログラミングフォント)

MS Storeからインストール

  • To-Do
  • Power BI Desktop
  • Ubuntu
  • Windows Terminal（超絶オススメ、コマンド系は一括で出来る）
  • A5:SQL Mk-2（ER図書く用）

Package Manager Chocolateyからインストール

Powershell管理者で以下のコマンド

# Powershell権限
Set-ExecutionPolicy RemoteSigned

# Proxy突破Powershell実行
.\Set-Proxy.ps1

# chocolatey インスコ
Set-ExecutionPolicy Bypass -Scope Process -Force; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))
#　バックアップファイルからソフトを一気にインストール
choco install choco.config -y

## 全体更新＆個別スキップ
choco upgrade all --except="'pkgname1, pkgname2'"

## 自動更新するソフトはChocoの管理外にする
choco uninstall googlechrome vscode --skip-autouninstaller
